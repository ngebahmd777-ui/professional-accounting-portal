## 🔧 نظام التكرار التلقائي

يتم تشغيل التكرار التلقائي عند فتح الصفحة:

- **تكرار يومي:** يتم إنشاء معاملة جديدة كل يوم للمستخدم
- **تكرار أسبوعي:** يتم إنشاء معاملة جديدة كل أسبوع
- المعاملات المتكررة تظهر بحالة **"قيد المراجعة"** حتى يقوم المدير بتأكيدها

---

## ✋ نظام الاعتراض

1. اضغط على زر **"اعتراض"** بجانب المعاملة أو الإنتاج
2. اكتب سبب الاعتراض
3. يتم إرسال رسالة واتساب إلى المدير على الرقم: `777524788`
4. يتم حفظ الاعتراض في النظام للمراجعة

---

## 🧵 إدارة إنتاج الخياطين

- **إضافة إنتاج:** المدير يضيف إنتاج الخياط بالكمية والموديل
- **السعر:** يمكن تركه فارغاً ويتم تحديده لاحقاً بعد الانتهاء
- **تأكيد:** بعد مراجعة الإنتاج، يقوم المدير بتأكيده أو تعديل السعر

## 💾 تخزين البيانات

جميع البيانات تُحفظ في **localStorage** للمتصفح:
- المعاملات المالية
- إنتاج الخياطين
- الأعضاء
- إعدادات التكرار التلقائي
- الاعتراضات

**ملاحظة:** البيانات خاصة بكل متصفح. للمشاريع الكبيرة يفضل استخدام قاعدة بيانات حقيقية.
